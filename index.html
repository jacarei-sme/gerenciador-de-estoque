Guia de Usuário: Sistema de Controle de Estoque

Este guia descreve como utilizar as funcionalidades do Sistema de Gerenciamento de Estoque feito para pelo Grupo 012 para o Departamento de Tecnologia da Informação da Secretaria Municipal de Educação de Jacareí.

Para acessar, você deve primeiro seguir para este link

Acesso ao Sistema.

Primeiro, para acessar o sistema, será necessário fazer o login.

Fonte: Autores.

Autenticação: Acesso a tela inicial do sistema. Para entrar, você deve fornecer primeiro seu e-mail e senha cadastrados.

Página Principal

Após o login, você será redirecionado a página principal, que é a central de operações do sistema. Conforme imagem abaixo:

Ações Principais: Logo após acesar a Página Principal, será apresentado as opções de acessar os Produtos, Categorias e Relatório.

Gerenciamento de Produtos

O sistema permite um controle quase completo sobre o cadastro dos produtos. Na imagem abaixo, temos a tela da Página de Produtos.

Fonte: Autores.

Adicionar Novo Produto

1. Na Tela de Produtos, clique em Adicionar Produto.

2. Será aberto um formulário onde deverá ser preenchido o nome, a descrição e a quantidade inicial do item.

3. Após preencher, clique em Salvar para que o produto seja adicionado ao estoque e apareça na lista principal.

Editar um Produto Existente

1. Na Tela de Produtos, localize o item que se deseja editar.

2. Clique na opção Editar.

3. O formulário de edição será exibido, permitindo a edição do nome ou da descrição do item.

4. Clique em Confirmar para salvar e atualizar as informações do produto.

Registro de Movimentações de Estoque

Aqui que esta a ideia principal do projeto, nesta parte é onde será feito as atualizações do estoque, podendo registrar se está sendo feito uma Saída ou uma Entrada do produto no estoque. A imagem abaixo é um exemplo da tela que contém o formulário para esse registro.

Fonte: Autores.

Registrar Entrada ou Saída

1. Na Tela de Produtos, localize o item que se deseja editar.

2. Clique no botão ao lado para selecionar qual movimentação deseja fazer: Entrada ou Retirada.

3. Uma janela será aberta, permitindo que se registre a movimentação.

4. Após preencher as informações, clique em Confirmar para registrar a movimentação.

O sistema oferece recursos para visualizar o histórico das movimentações feitas pelos usuários, facilitando o controle. Abaixo esta uma imagem demontrando como será exibido as informações.

Informações do relatório

Atualmente, o relatório apenas gera uma tela demonstrando todas as movimentações feitas pelos usuários. As informações apresentadas na tela são:

- Data da movimentação. (Atualmente a hora é demonstrada em UTC)

- Produto movimentado.

- Tipo da movimentação. (Entrada ou Saída)

- Quantidade.

- Usuário que fez a operação.

- Justificativa.
